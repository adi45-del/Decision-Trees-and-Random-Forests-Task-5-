# Decision-Trees-and-Random-Forests-Task-5-
AI/ML Internship Task 5 – Decision Tree and Random Forest for Heart Disease Classification.

## Objective
The objective of this task is to learn and implement **Decision Tree** and **Random Forest** algorithms for classification and compare their performance.

## Dataset Used
- **Heart Disease Dataset**
- The dataset contains patient health information used to predict whether a person has heart disease.

## Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Tasks Performed
- Imported and preprocessed the Heart Disease dataset.
- Split the dataset into training and testing sets.
- Trained a Decision Tree Classifier.
- Visualized the Decision Tree.
- Controlled overfitting by adjusting the tree depth.
- Trained a Random Forest Classifier.
- Compared the accuracy of both models.
- Analyzed feature importance.
- Evaluated model performance using Cross-Validation.

## Learning Outcome
Through this task, I learned:
- Decision Tree Classification
- Random Forest Classification
- Overfitting and Tree Depth
- Feature Importance
- Cross-Validation
- Model Performance Comparison
